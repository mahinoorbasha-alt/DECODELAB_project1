🌾 Farmer Portal
📌 Project Overview
Farmer Portal is a responsive and user-friendly web application developed to support farmers by providing essential agricultural information in a single platform. The portal helps farmers register their details, explore crop information, view market prices, learn about government schemes, and access useful farming guidance.
The project is built using HTML, CSS, and JavaScript and focuses on creating an interactive and informative experience for users.
✨ Features
🔐 Login System
Attractive login page with animated background.
Simple authentication interface for accessing the portal.
🏠 Home Dashboard
Welcome section with motivational agricultural content.
Interactive cards for:
Farming Tips
Irrigation Guide
Organic Farming
Detailed information displayed when each card is selected.
👨‍🌾 Farmer Registration
Register farmer details including:
Farmer Name
Mobile Number
Village
District
Crop Type
Land Area
Data stored using browser Local Storage.
Success confirmation after registration.
🌾 Crop Information
Search functionality for finding crops quickly.
Detailed information about:
Rice
Wheat
Cotton
Maize
Sugarcane
Millets
Barley
Displays cultivation requirements, uses, and benefits.
💰 Market Prices
Displays market prices of various crops.
Easy-to-read table format.
🏛 Government Schemes
Interactive scheme cards.
